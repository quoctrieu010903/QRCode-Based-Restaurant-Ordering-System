# QR Code-Based Restaurant Ordering System

A restaurant ordering system that allows customers to scan a QR code at their table, browse the menu, place orders, and track order status digitally. The system is designed to improve customer experience, reduce manual workload for staff, and streamline restaurant operations.

## Project Overview

This project provides a QR-based self-ordering workflow for restaurants:

- Scan table QR code
- Create table session
- Browse menu
- Add items to cart
- Place orders
- Track order status in real time

The system helps restaurants modernize the ordering process and improves communication between customers and kitchen staff.

## Main Features

- QR code-based table access
- Table session creation and management
- Digital menu browsing
- Cart and order placement
- Real-time order status synchronization
- Order monitoring support
- Media upload and storage for menu items

## Back-end Scope

The back-end system is responsible for:

- Building RESTful APIs for table session, menu, cart, ordering, and monitoring modules
- Managing database schema and entity relationships
- Handling order processing workflow
- Synchronizing order status in real time between customer and kitchen screens
- Supporting media integration and storage
- Improving system reliability with validation and exception handling
